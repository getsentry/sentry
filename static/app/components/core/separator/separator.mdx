import APIReference from '!!type-loader!sentry/components/core/separator';

export const types = {Separator: APIReference.Separator};

The `Separator` component creates visual divisions between content areas. It renders as an HTML `<hr>` element with flexible styling options and supports both horizontal and vertical orientations.

## Basic Usage

The Separator component requires an `orientation` prop to determine its layout direction.

```jsx
// Horizontal separator (default styling)
<Separator orientation="horizontal" border="primary" />

// Vertical separator
<Separator orientation="vertical" border="primary" />
```

## Horizontal Separators

Horizontal separators are ideal for dividing content vertically, such as between sections or list items.

```jsx
<Flex direction="column" gap="md">
  <div>Content above separator</div>

</Flex>
```
