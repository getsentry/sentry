Use `AvatarButton` when the avatar itself should be the clickable control.

## Basic Usage

```jsx
<AvatarButton
  aria-label="Open profile"
  onClick={() => {}}
  avatar={{type: 'letter_avatar', identifier: 'user-1', name: 'Jane Doe'}}
/>
```

## Avatar Definitions

The `avatar` prop accepts a typed avatar descriptor. Only the data fields are used —
`type`, `identifier`, `name`, and any type-specific fields (`uploadUrl`, `gravatarId`).
Style props such as `suggested` or `round` are controlled by the button itself.

### averageSampler

Mean RGB of all opaque pixels. Fast but blends all colors together.

<Storybook.Demo>
  {IMAGES.map(img => (
    <AvatarButton
      key={img.key}
      sampler={averageSampler}
      aria-label={img.name}
      avatar={{type: 'upload', identifier: img.key, name: img.name, uploadUrl: img.uploadUrl}}
    />
  ))}
</Storybook.Demo>

### saturatedAverageSampler (default)

Mean of chromatic pixels only. Skips near-white/gray/black — returns null for neutral images.

<Storybook.Demo>
  {IMAGES.map(img => (
    <AvatarButton
      key={img.key}
      sampler={saturatedAverageSampler}
      aria-label={img.name}
      avatar={{type: 'upload', identifier: img.key, name: img.name, uploadUrl: img.uploadUrl}}
    />
  ))}
</Storybook.Demo>

### dominantBucketSampler

Most-populated 2-bit-per-channel histogram bucket. Finds the single most common hue region.

<Storybook.Demo>
