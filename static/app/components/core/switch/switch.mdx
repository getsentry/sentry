To create a basic switch, use the `<Switch>` component with `checked` and `onChange` props. The switch doesn't include a label, so you'll need to provide your own accompanying label using HTML `<label>` element for accessibility.

```jsx
<Switch />
```

## Sizes

Switches are available in two sizes: `sm` (default) and `lg`.

## Controlled or Uncontrolled State

Switches support controlled or uncontrolled state. To use a controlled state, pass the `checked` and `onChange` props. To use an uncontrolled state, pass the `defaultChecked` prop.

```jsx
// Controlled Switch
<Switch checked={checked} onChange={handleChange} />
// Uncontrolled Switch
<Switch defaultChecked />
```
