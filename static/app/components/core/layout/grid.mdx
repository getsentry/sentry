The `Grid` component is a layout component that extends the `Container` component with CSS grid properties.

## Basic Usage

To create a basic grid container, wrap elements in `<Grid>` and define columns using `columns`.

```jsx
<Grid columns="repeat(3, 1fr)" gap="md">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Grid>
```

### Alignment

Grid provides fine-grained control over alignment using `justify`, `align`, `justifyContent`, and `alignContent`.

<Storybook.Demo>
  <Grid columns="repeat(2, 1fr)" gap="md" padding="md">
    {[
      {justifyContent: 'start', align: 'start', label: 'start/start'},
      {justifyContent: 'center', align: 'center', label: 'center/center'},
      {justifyContent: 'end', align: 'end', label: 'end/end'},
      {justifyContent: 'stretch', align: 'stretch', label: 'stretch/stretch'},
    ].map(({justifyContent, align, label}) => (
      <Grid
        key={label}
        columns="repeat(2, 1fr)"
        gap="sm"
        justifyContent={justifyContent}
        align={align}
        border="primary"
      >
        <div>{label}</div>
        <div>Item</div>
      </Grid>
    ))}
  </Grid>
</Storybook.Demo>
```jsx
<Grid columns="repeat(2, 1fr)" justifyContent="center" align="center">
  <div>Centered Item</div>
  <div>Another Item</div>
</Grid>
```

### Responsive Props

All props support responsive values using breakpoint objects. Breakpoints are: `xs`, `sm`, `md`, `lg`, `xl`, `2xl`.

Example of a responsive grid container that changes from single column on small screens to multiple columns on larger screens.

  columns={{xs: '1fr', sm: 'repeat(2, 1fr)', md: 'repeat(3, 1fr)'}}
  gap="md"

If a prop is not specified for a breakpoint, the value will **not** be inherited from the previous breakpoint.
