The `Surface` component is a simplified layout component built on top of the `Container` component. It provides a focused API for surfaces with only the essential props: `variant`, `elevation`, and `radius`.

To change the layout of `Surface` children, wrap children with a specific layout component (such as [`Flex`](/stories/core/flex), [`Stack`](/stories/core/stack), [`Grid`](/stories/core/grid)).

## Variants

### Primary

Use `variant="primary"` for primary content areas.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1}>
    <Surface variant="primary">Primary surface</Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="primary">Primary surface</Surface>
```

### Secondary

Use `variant="secondary"` for secondary content areas.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1}>
    <Surface variant="secondary">Secondary surface</Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="secondary">Secondary surface</Surface>
```

### Tertiary

Use `variant="tertiary"` for tertiary content areas.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1}>
    <Surface variant="tertiary">Tertiary surface</Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="tertiary">Tertiary surface</Surface>
```

### Overlay

Use `variant="overlay"` for floating surfaces like dropdowns, modals, and popovers. Overlay surfaces include a border and box-shadow.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1}>
    <Surface variant="overlay">Overlay surface (default high elevation)</Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="overlay">Overlay surface</Surface>
```

## Elevation

The `elevation` prop is only available for `variant="overlay"`. It controls the shadow intensity.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1} gap="md">
    <Surface variant="overlay" elevation="low">
      Low elevation
    </Surface>
    <Surface variant="overlay" elevation="high">
      High elevation (default)
    </Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="overlay" elevation="low">
  Low elevation
</Surface>
<Surface variant="overlay" elevation="high">
  High elevation (default)
</Surface>
```

## Radius

The `radius` prop controls the border radius. Overlay surfaces default to `md` radius; other variants have no radius by default.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1} gap="md">
    <Surface variant="primary" radius="sm">
      Small radius
    </Surface>
    <Surface variant="primary" radius="md">
      Medium radius
    </Surface>
    <Surface variant="primary" radius="lg">
      Large radius
    </Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="primary" radius="sm">
  Small radius
</Surface>
<Surface variant="primary" radius="md">
  Medium radius
</Surface>
<Surface variant="primary" radius="lg">
  Large radius
</Surface>
```
