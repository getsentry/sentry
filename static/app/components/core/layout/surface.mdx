The `Surface` component is a simplified layout component built on top of the `Container` component. It provides a focused API for surfaces with only the essential props: `variant`, `elevation`, and `radius`.

To change the layout of `Surface` children, wrap children with a specific layout component (such as [`Flex`](/stories/core/flex), [`Stack`](/stories/core/stack), [`Grid`](/stories/core/grid)).

## Flat Variants

Use basic variants for content areas: `primary` for main content, `secondary` for supporting content, and `tertiary` for nested or de-emphasized areas.

## Overlay Variant & Elevation

Use `variant="overlay"` for floating surfaces like dropdowns, modals, and popovers. Overlay surfaces include a border and box-shadow. The `elevation` prop controls shadow intensity and is only available for overlay surfaces.

<Storybook.Demo>
  <Stack minWidth="0" flexGrow={1} gap="md">
    <Surface variant="overlay" elevation="low">
      <Flex padding="xl lg">
        <Text variant="primary">

        </Text>
      </Flex>
    </Surface>
    <Surface variant="overlay" elevation="high">
      <Flex padding="xl lg">Overlay: High elevation</Flex>
    </Surface>
  </Stack>
</Storybook.Demo>
```jsx
<Surface variant="overlay" elevation="low">

</Surface>
<Surface variant="overlay" elevation="high">
  Overlay: High elevation
</Surface>
```
