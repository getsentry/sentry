This project is a monorepo of a Python Django project (found in `src/`) and a React frontend project (found in `static`). Follow the best practices and guidelines when working in each respective project.

We also use:
- Postgresql as the database
- `devservices` to manage our Docker container runtimes
- `devenv` to manage our development environment
- `api-docs/` for OpenAPI specifications

ALWAYS make sure your changes pass our lint and pre-commit checks.
